# Week 2 Responsive Web Design Report

## Objective

The objective of this project was to create a responsive webpage that adapts smoothly across desktop, tablet, and mobile devices using HTML and CSS.

## Technologies Used

* HTML5
* CSS3
* Flexbox
* Media Queries

## Features

* Responsive navigation bar
* Hero section
* Services section
* Responsive image gallery
* Contact section
* Footer
* Smooth scrolling
* Mobile-friendly layouts

## Responsive Design

Media queries were used to adjust layouts, images, spacing, and text sizes for different screen sizes. Flexbox was implemented to create flexible and scalable layouts.

## Challenges Faced

Maintaining responsiveness across different devices and screen resolutions was challenging. These issues were solved using responsive CSS techniques and flexible layouts.

## Conclusion

This project improved my understanding of responsive web design, media queries, flexible grids, and frontend development practices.
